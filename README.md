# engine-sim-mod

**->Keybinds<-**
U -> Toggle Supercharger
T -> Switch Transmission (Automatic / Manual)
J -> Switch Transmission mode ( Comfort / Sport )

**->Defaults<-**
Supercharger -> Off
Transmission -> Manual
Mode -> Comfort

# How to install?
Download this Repository and run the `engine-sim-app.exe` in the `bin` folder.
